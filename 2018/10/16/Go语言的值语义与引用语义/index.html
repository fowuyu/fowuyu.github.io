Go语言的值语义与引用语义

任何一门语言，如果其有指针概念，则需要了解这门语言是否拥有引用语义，当然你可以说任何引用语义都是值语义（正确的废话）。之所以这个内容需要深入理解，因为它涉及到一个重要的内容，那就是比较 ，编程语言的三大结构，都需要依赖比较功能，而比较就需要我们理解比较的到底是变量的值，还是变量指向的值。

func main{
    a := "yichen"
    b := "yichen"
    fmt.Print(a == b) //true 可以看出string是值语义
}

func main{
    a := [3]{1,2,3}
    b := a
    b[1]++
    fmt.Print(a,b) //{1,2,3},{1,3,3} 可以看出数组居然也是值语义
}

func main{
    a := []{1,2,3}
    b := a
    b[1]++
    fmt.Print(a,b) //{1,3,3},{1,3,3} 可以看出slice是引用语义
}

type Person struct{
    age int
    name string
}

func main() {
    yichen := Person{21,"yichen"}
    pengfan := Person{21,"yichen"}
    fmt.Println(yichen == pengfan) //true 不可思议的是struct也是值语义
}



type Person struct{
    age int
    name string
}
func main() {
    a := Person{21,"yichen"}
    b := &a
    c := &a
    fmt.Println(b == c) //true 指针也是值语义
}

总结

值语义

基本类型，如byte、int、bool、float32、float64和string等;
复合类型，如数组(array)、结构体(struct)和指针(pointer)等。

引用语义

slice,map,list,heap,ring,channels
